# Changelog

## [1.0.2] - 2026-04-04

- Added support for Minecraft `26.1.1` / Paper `26.1.1.build.14-alpha`.
- Added bStats metrics reporting support.
- Removed the deprecated `HIDE_ITEM_SPECIFICS` item flag because it is no longer available on newer Paper builds.

## [1.0.1] - 2026-03-29

- Added a GitHub release version checker that periodically looks for newer plugin versions.
- Added join-time update notifications for server operators with a direct download link when an update is available.

## [1.0.0] - 2026-03-27

- First release of Majestic-Spawners.
